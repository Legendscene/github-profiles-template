**50+ stunning, ready-to-use GitHub profile README templates.**  
Just **copy-paste** → replace `yourusername` → done! No build tools, no CSS.

---

### ⚡ Quick Start

```bash
# 1. Create a repo named exactly like your GitHub username
# 2. Add a README.md
# 3. Copy any template from this repo
# 4. Replace every `yourusername` with your actual username
# 5. Push — your profile is live! 🎉
```

## 🚦 How to Use

1. **Fork this repo** or copy any template file
2. Go to your profile repo → `username/username` → `README.md`
3. Paste the template content
4. Find & replace `yourusername` with your GitHub username
5. Customize badges, links, and text
6. Commit & push — **instant glow-up** ✨
