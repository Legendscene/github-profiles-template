## 💡 Pro Tips

1. **Dynamic rating badges**: Use `https://img.shields.io/badge/dynamic/json?url=https://cp-logo.vercel.app/codeforces/torvalds&label=Codeforces&color=445f9d` for auto-updating ratings via [CP-Logo API](https://cp-logo.vercel.app/).

2. **LeetCode card**: Add a LeetCode stats card: `![LeetCode Stats](https://leetcard.jacoblin.cool/yourusername?theme=dark&font=Inter)`.

3. **Rank colors matter**: Use the official rank colors — Codeforces Legendary = red/orange, Guardian = yellow, etc. Badge colors should match.

4. **Problem count signals**: 500+ = decent, 1000+ = dedicated, 2500+ = elite. Be honest — this is easy to verify.

5. **Global rank**: If you've ever placed top 100 in any contest, put it in the stats block. That single number carries more weight than your rating.
