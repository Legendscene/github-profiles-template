## 🔧 Complete Customization Guide

### Step 1: Basic Information
| What to Replace | Where It Appears | Replace With | Example |
|----------------|-----------------|--------------|---------|
| `<your-name>` | Below avatar | Your full name | Emily Zhang |
| `yourusername` | ALL URLs, stats, avatar, project links | Your GitHub username | emilyzhang |
| `yourportfolio.com` | Portfolio badge link | Your portfolio URL | https://emilyzhang.dev |

### Step 2: Social Links
Replace these badge URLs:
- `https://linkedin.com/in/yourusername` → `https://linkedin.com/in/YOUR_LINKEDIN`
- `https://codepen.io/yourusername` → `https://codepen.io/YOUR_HANDLE`
- `https://dribbble.com/yourusername` → `https://dribbble.com/YOUR_HANDLE`
- `https://yourportfolio.com` → Your portfolio URL

### Step 3: Frontend Arsenal (Skill Icons)
The `skillicons.dev` URL shows framework/tool icons. Customize:
- Change the comma-separated list: `i=react,next,vue,angular,svelte,js,ts,html,css,sass,tailwind,bootstrap,materialui,figma,webpack,vite`
- Find available icons at `https://skillicons.dev/`
- `perline=8` = icons per row. Adjust based on your count.

### Step 4: Stats
Replace `yourusername` in all three stats URLs:
- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-stats.vercel.app/api/top-langs/?username=yourusername&...`
- `https://github-readme-streak-stats.herokuapp.com/?user=yourusername&...`

### Step 5: Skills Table
The table lists categories and technologies. Customize:
- Change category names (e.g., add "**Animations**", "**Backend**" if you do full-stack)
- Replace the technology lists with what you actually use
- Add or remove rows by copying/removing `| **Category** | Tech1, Tech2 |` lines

### Step 6: UI Showcase
Update project links and descriptions:
- `https://github.com/yourusername/ui-library` → your actual repo
- Change descriptions from "Reusable React components" to what you built
- Add more rows by duplicating the `<tr>` pattern

### Step 7: Typing SVG Animation
Customize the header animation:
- `lines=Frontend+Developer;UI+%2F+UX+Enthusiast;Pixel+Perfect+%F0%9F%8E%A8` — semicolons separate each line
- `color=F75C7E` — change the hex color for your brand
- `font=Fira+Code` — try `font=JetBrains+Mono`, `font=Source+Code+Pro`

## 💡 Pro Tips

1. **Radical theme**: The `theme=radical` gives a pink/red vibe that suits frontend. Try `theme=synthwave` (retro purple/pink) or `theme=jolly` (warm colors) for similar aesthetics.
2. **Skill icons sizing**: The `skillicons.dev` icons are small but crisp. For larger icons, use Shields.io badges instead with `style=for-the-badge&logo=react`.
3. **Codepen/Dribbble badges**: These platforms show your design work and code experiments. If you don't use them, replace with Behance, Figma Community, or your design blog.
4. **Skills table depth**: Add a "**Performance**" or "**Accessibility**" category row — it shows you care about best practices (a11y, Core Web Vitals, etc.).
5. **Footer quote**: The footer quote is URL-encoded in the capsule-render URL. Change it at `https://www.urlencoder.org/` to your favorite design quote.

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with your actual GitHub username
5. Replace `<your-name>`, `yourportfolio.com`
6. Update skill icons, skills table, and UI showcase with your real projects
7. Commit — your new profile is LIVE! 🎉
