Replace these badge URLs:
- `https://yourusername.itch.io` → `https://YOUR_HANDLE.itch.io`

- Engines: `Unity`, `Unreal Engine`, `Godot`, `GameMaker`, `RPG Maker`, `Construct`
- Art: `Blender`, `Aseprite`, `Photoshop`, `Krita`, `Spine`
- Audio: `FMOD`, `Wwise`, `Audacity`, `Bosca Ceoil`
- Remove engines/tools you don't use, add ones you do

### Step 5: Stats

- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-streak-stats.herokuapp.com/?user=yourusername&...`

### Step 6: My Games Section
Update each game in the table:
- Change game titles from "Super Platformer" to your actual game names
- Add real descriptions

- To add more games, duplicate the `<td width="50%">` block (max 2 per row, add another `<tr>` for more rows)

### Step 7: Achievements
Replace the placeholder achievements with your real ones:
- 🥇 Game jam wins (Ludum Dare, GMTK, Global Game Jam)

- Format: `- 🥇 [Achievement] — [details]`

- The `font=Press+Start+2P` gives the 8-bit retro look
- Change the lines: `lines=Game+Developer;Indie+Game+Dev;Pixel+Pusher`
- `color=FF6B35` is the orange color — change to your game's brand color

## 💡 Pro Tips

1. **Gruvbox theme**: The `theme=gruvbox` (warm orange/brown) perfectly matches the game dev vibe. Try `theme=maroongold`, `theme=greatescape`, or `theme=prussian` for other warm themes.
2. **Itch.io integration**: Itch.io pages are free and essential for indie game devs. Even if you only have prototypes, linking them shows you actually build games.
3. **Game tables**: Keep descriptions short (5-10 words). Add a "Status" column or a "Wishlist on Steam" badge for upcoming releases.
4. **8-bit font**: The `Press Start 2P` font is authentic but only looks good at `size=22` or smaller. If the text overflows, reduce font size or shorten the lines.
5. **Avatar shape**: `border-radius:20%` gives slightly rounded corners (more "pixel-like"). Change to `border-radius:0` for sharp squares (pixel art style) or `50%` for circles.

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with your actual GitHub username
5. Replace `<your-name>`, `yourgame`, and itch.io URLs
6. Update engines, game table, and achievements with your real info
7. Commit — your new profile is LIVE! 🎉
