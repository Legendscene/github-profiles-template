## ΏθΤκ Pro Tips

1. **Use `animation=twinkling` on the capsule header** έΑΦ It gives the header a subtle star-like twinkle effect that perfectly matches the galaxy theme. Don't change it to `blinking` έΑΦ that's too aggressive for a space theme.
2. **Match the deep purple background** έΑΦ The stats use `bg_color=0d0221` (near-black with purple tint). Always use this or similar on any custom elements to maintain the cosmic darkness.
3. **Add real star constellations** έΑΦ ASCII stars in the quote section look good, but you can also create actual star patterns using Unicode: use `έεο` `έαΖ` `έαΗ` `έεπ` in varying patterns to simulate constellations.
4. **Name repos with cosmic names** έΑΦ If you can, give your repos space-themed names: "nebula", "supernova", "constellation", "asteroid", "orbit". It makes the Featured Constellations section much more immersive.
5. **Change the capsule header text** έΑΦ Replace `COSMIC DEVELOPER` with your own brand: `STAR CODER`, `SPACE DEV`, or `GALACTIC ENGINEER`. Keep it uppercase for the cosmic vibe.
6. **Portfolio badge icon** έΑΦ Use `logo=starship` or `logo=space` or `logo=astro` for the portfolio badge. If none work, `logo=link` is a safe fallback that still looks good with purple.
7. **Add a visitor map** έΑΦ For a fun extra, add a GitHub visitors map widget showing where your visitors come from. It fits the "exploration" theme perfectly.
