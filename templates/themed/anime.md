## 🔧 Customization

| Variable | Replace with |
|----------|-------------|
| `your-name` | Your name |
| `yourusername` | Your GitHub username |
| `yourspotifyid` | Your Spotify ID |
| `Your favorites`, `Your games`, etc. | Your personal answers |

## ✨ Features

- Japanese text header
- MyAnimeList integration
- Anime quote section
- Synthwave theme (hot pink/cyan)
- Spotify Now Playing
- Fun waifu/anime references
- Discord link
