## 💡 Pro Tips

1. **Never change the green** — The Matrix theme relies on `#00ff41` (phosphor green). Every badge, stat card, and accent should use this exact shade. Keeping `bg_color=000000` (pure black) ensures maximum contrast.
2. **Use Matrix movie quotes** — The quotes are what sell the theme. Use iconic lines: "There is no spoon", "I know kung fu", "Unfortunately, no one can be told what the Matrix is", "Free your mind". Avoid non-Matrix quotes.
3. **The ASCII "MATRIX" logo is critical** — It spells out MATRIX in block ASCII. If you change the name, regenerate it at [patorjk.com](https://patorjk.com/software/taag/) using the "ANSI Shadow" or "Bloody" font.
4. **Add a digital rain GIF** — Find a Matrix "digital rain" GIF from Giphy and embed it at the top or bottom of your profile. Use `![Digital Rain](https://media.giphy.com/media/...)` with a width limit.
5. **Use `theme=matrix` for trophies** — This is a specific theme on the trophy API that renders in Matrix green. Very few profiles use it, which makes yours stand out.
6. **The "Follow the white rabbit" badge** — This is a signature element. Don't remove it. The icon uses `logo=matrix` which shows a subtle Matrix reference in the badge.
7. **Keep the "System Access" label** — The visitor counter label `System+Access` fits the theme perfectly. Alternatives: `Zion+Access`, `Node+Connections`, `Terminal+Sessions`.
