## 💡 Pro Tips

1. **Snake animation setup**: The snake grid requires a GitHub Action. Create `.github/workflows/snake.yml` that generates `output/github-contribution-grid-snake.svg` using `Platane/snk`. Without this, the image will 404.

2. **Multiple typing lines**: Add up to 10 lines in the typing SVG by separating with `;` and URL-encoding spaces as `+` or `%20`. Each line pauses for `duration` ms — use `&duration=2000&pause=1000` for better readability.

3. **Transparent background**: The `bg_color=00000000` makes stats transparent so they overlay nicely on profile backgrounds. If you use a dark profile theme, swap to a solid background like `bg_color=1a0b2e`.

4. **GIF alternatives**: If the coding GIF doesn't load, use a local or shorter URL: `https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif`. Test that the URL works before committing.

5. **Header text animation**: Change `animation=twinkling` to `animation=fadeIn` or `animation=blinking` in the capsule-render URL for different header effects. `animation=scaleIn` grows the text on load.

6. **Custom capsule colors**: Replace `color=timeGradient` with static gradient: `color=0:ff6ec7,100:833ab4` for a pink-to-purple gradient that matches the synthwave theme.
