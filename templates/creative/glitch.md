## ūüí° Pro Tips

1. **Glitch font alternatives**: The template uses `Share+Tech+Mono` ‚ÄĒ a free Google Font. Swap to `VT323` for a more retro terminal feel, or `Major+Mono+Display` for a chunkier glitch look. Just change the `font=` parameter.

2. **Glitch CSS animation**: Add `<style>.glitch-text { animation: glitch 1s infinite; } @keyframes glitch { 0% { text-shadow: 2px 0 #ff0055, -2px 0 #00ffff; } 50% { text-shadow: -2px 0 #ff0055, 2px 0 #00ffff; } 100% { text-shadow: 2px 0 #ff0055, -2px 0 #00ffff; } }</style>` before your name for a true glitch effect.

3. **Duotone color scheme**: The hot pink (`#ff0055`) + cyan (`#00ffff`) combo is core to the glitch aesthetic. Swap them on alternate badges: even badges = pink, odd badges = cyan. Keep this pattern consistent.

4. **ASCII art generator**: Use `https://patorjk.com/software/taag/` with the "ANSI Shadow" font to generate your name in glitchy ASCII art. Replace the current block with your own name rendered in the same style.

5. **Error code easter eggs**: Use inside-joke error codes as footer messages: `0xG00D` instead of `0xDEADBEEF`, or `ERR0R:42` for a Hitchhiker's Guide reference. Makes the profile memorable.

6. **Capsule divider**: The `type=rect&height=3` creates a thin line divider. Change `height=3` to `height=1` for a subtle hairline, or `height=10` for a thick color bar. The gradient `0:ff0055,100:00ffff` creates the duotone effect.
