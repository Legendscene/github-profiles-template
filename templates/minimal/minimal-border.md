## 🔧 Complete Customization Guide

### Step 1: Basic Information
| What to Replace | Where It Appears | Replace With | Example |
|----------------|-----------------|--------------|---------|
| `<your-name>` | Typing SVG, table header, left card | Your full name | Maria Garcia |
| `yourusername` | ALL URLs, stats, avatar, social links | Your GitHub username | mariagarcia |
| `<your-tagline>` | Left card below name | Your role/title | Full Stack Developer |
| `<your-location>` | Left card with 📍 | Your city, country | Madrid, Spain |
| `[project]` | Right card bullet | Current project | an AI chatbot |
| `[tech]` | Right card bullet | What you're learning | Rust |
| `[topics]` | Right card bullet | Your expertise | React, Node.js |
| `your@email.com` | Right card email link | Your email | maria@example.com |
| `[fun fact]` | Right card bullet | Fun fact about you | I speak 3 languages |

### Step 2: Social Links
Replace these in the social badges section:
- `https://linkedin.com/in/yourusername` → `https://linkedin.com/in/YOUR_LINKEDIN`
- `https://twitter.com/yourusername` → `https://twitter.com/YOUR_HANDLE`
- `https://discord.gg/yourusername` → Your Discord server invite link
- `https://medium.com/@yourusername` → `https://medium.com/@YOUR_HANDLE`

### Step 3: Typing SVG Animation
The typing animation cycles through messages. Customize:
- `lines=Hi+%F0%9F%91%8B%2C+I'm+<your-name>;I+build+cool+stuff!` — each semicolon-separated value is a line that types in sequence
- `color=36BCF7` — change the hex color to match your brand
- `pause=1000` — milliseconds each line stays visible

### Step 4: Skill Icons
The `skillicons.dev` URL shows icons. Change the comma-separated list:
- `https://skillicons.dev/icons?i=js,ts,react,nextjs,nodejs,py,django,rust,go,docker,k8s,aws,gcp,git,github,linux,vscode&perline=8`
- Find available icon names at `https://skillicons.dev/`
- `perline=8` controls icons per row — adjust based on how many you have

### Step 5: Stats, Activity Graph & Followers
Replace `yourusername` in ALL these:
- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-streak-stats.herokuapp.com/?user=yourusername&...`
- `https://github-readme-activity-graph.vercel.app/graph?username=yourusername&...`
- `https://komarev.com/ghpvc/?username=yourusername&...`

## 💡 Pro Tips

1. **Two-column stats width**: The stats and streak cards use `width="49%"` to sit side-by-side. On mobile they'll stack. Change to `width="45%"` with some margin if they look cramped.
2. **Typing SVG customization**: Add more lines by adding more semicolons. Example: `lines=Full+Stack+Developer;Open+Source+Lover;Coffee+Addict` — URL-encode spaces as `+` or `%20`.
3. **Activity graph theme**: Change `theme=react-dark` to any of: `github`, `github-dark`, `github-light`, `tokyonight`, `dracula`, `merko`. Find the full list at the repo.
4. **Make it yours**: Add a "Currently listening to" or "Recent activity" section between stats. GitHub readme is flexible HTML.
5. **Border radius effect**: The `border_radius=10` in stats URLs adds rounded corners. Increase to 20 for more rounding, or remove for sharp corners.

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with actual username
5. Replace `<your-name>`, `<your-tagline>`, `<your-location>`, `your@email.com`, and bracket placeholders
6. Customize the skill icons list and social links
7. Commit — your new profile is LIVE! 🎉
