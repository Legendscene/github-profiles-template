## 🔧 Complete Customization Guide

### Step 1: Basic Information
| What to Replace | Where It Appears | Replace With | Example |
|----------------|-----------------|--------------|---------|
| `<your-name>` | Main heading | Your full name | John Doe |
| `yourusername` | ALL URLs, stats cards, social links | Your GitHub username | johndoe |
| `somewhere` | Subtitle tagline | Your location | San Francisco, CA |
| `[current project]` | About Me section | What you're building | an AI chatbot |
| `[technology]` | About Me section | What you're learning | Rust and WebAssembly |
| `[open source project]` | About Me section | Project you want collaboration on | a blockchain framework |
| `[topics]` | About Me section | Your expertise areas | React, Node.js, Python |
| `[email]` | About Me section | Your contact email | john@example.com |
| `[fun fact]` | About Me section | A personal fun fact | I can solve a Rubik's cube in 30s |

### Step 2: Social Links
Find these lines and replace the URLs:
- `https://twitter.com/yourusername` → `https://twitter.com/YOUR_HANDLE`
- `https://linkedin.com/in/yourusername` → `https://linkedin.com/in/YOUR_LINKEDIN`
- `https://github.com/yourusername` → `https://github.com/YOUR_USERNAME`
- `https://dev.to/yourusername` → `https://dev.to/YOUR_HANDLE`
- `https://yourwebsite.com` → Your portfolio URL

### Step 3: GitHub Stats
Replace `yourusername` in these URLs:
- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-streak-stats.herokuapp.com/?user=yourusername&...`
- `https://github-readme-stats.vercel.app/api/top-langs/?username=yourusername&...`

These will auto-generate real stats once you use your actual username.

### Step 4: Snake Animation
The snake grid shows your contribution activity. To enable it:
1. Go to Actions → New workflow → Set up a workflow yourself
2. Paste the GitHub Snake action YAML (search `github-snake` action in Marketplace)
3. Commit — the snake will auto-generate after first run

### Step 5: Header & Footer Text
- Change `Hey%20There!` (URL-encoded = "Hey There!") in the capsule-render URL to your own greeting
- Use `https://www.urlencoder.org/` to encode any text

## 💡 Pro Tips

1. **Color customization**: Change the `color=0:0d1117,100:58a6ff` values in capsule-render URLs to match your brand — use any hex color code.
2. **Disable the snake**: If you don't set up the GitHub Action, the snake image will 404. Either set it up or remove that section entirely.
3. **Stats card sizing**: Use the `height` parameter (e.g. `height="150"`) to control card heights. Adjust for perfect alignment.
4. **Add more badges**: Duplicate the social badge pattern (`<a href="..."><img src="..." /></a>`) to add YouTube, Medium, Discord, or any other platform.
5. **Hide/reveal details**: Change `<details open>` to `<details>` to make the About section collapsed by default.

---

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with your actual GitHub username
5. Repeat for `your-name`, `somewhere`, and bracket placeholders
6. Commit the changes — your new profile is LIVE! 🎉
