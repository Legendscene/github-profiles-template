## 🔧 Customization

| Variable | Replace with |
|----------|-------------|
| `your-name` | Your display name |
| `yourusername` | Your GitHub username |
| `somewhere` | Your location |
| `[email]` | Your email address |
| `[cool project]`, `[technology]`, etc. | Your details |

## ✨ Features

- Dark theme (Tokyonight)
- GitHub stats + top languages
- Streak stats
- Animated snake contribution grid
- Social links
- Expandable About Me section
- Capsule wave header/footer
