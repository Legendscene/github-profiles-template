## 🔧 Customization

| Variable | Replace with |
|----------|-------------|
| `your-name` | Your display name |
| `yourusername` | Your GitHub username |
| `your-city`, `your-country` | Your location |
| `your@email.com` | Your email |
| `job-title` | Your current role |

## ✨ Features

- Clean white theme (default GitHub theme)
- YAML-style bio
- GitHub stats
- Streak stats
- Projects table
- Compact social badges
- Visitor counter
