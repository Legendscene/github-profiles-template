## 🔧 Complete Customization Guide

### Step 1: Basic Information
| What to Replace | Where It Appears | Replace With | Example |
|----------------|-----------------|--------------|---------|
| `<your-name>` | Main heading & YAML bio | Your full name | Sarah Chen |
| `yourusername` | ALL URLs, stats cards, social links | Your GitHub username | sarahchen |
| `<your-city>` | YAML bio location | Your city | Berlin, Germany |
| `<your-country>` | YAML bio location | Your country | Germany |
| `<job-title>` | YAML bio job field | Your current role | Full Stack Developer |
| `your@email.com` | Email badge link | Your email | sarah@example.com |

### Step 2: Social Links
Replace these URLs in the social badge section:
- `https://twitter.com/yourusername` → `https://twitter.com/YOUR_HANDLE`
- `https://linkedin.com/in/yourusername` → `https://linkedin.com/in/YOUR_LINKEDIN`
- `https://stackoverflow.com/users/yourusername` → `https://stackoverflow.com/users/YOUR_ID/YOUR_NAME`

### Step 3: YAML Bio Section
Replace the entire YAML block with your real info:
- **technologies**: List what you actually use (aim for 3-6 items)
- **interests**: List what you care about (aim for 3-4 items)
- Feel free to add new fields — `education`, `hobbies`, `goals`

### Step 4: GitHub Stats
Just find & replace `yourusername` in these URLs:
- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-streak-stats.herokuapp.com/?user=yourusername&...`
- `https://github-readme-stats.vercel.app/api/top-langs/?username=yourusername&...`

### Step 5: Projects Table
Replace each project row:
- `https://github.com/yourusername/project1` → actual repo URL
- `Brief description` → what the project does in 2-5 words
- Stars column → add real star counts or remove stars column

### Step 6: Header Text
Change `Hello!` in the capsule URL to your own text. URL-encode it first at `https://www.urlencoder.org/`

## 💡 Pro Tips

1. **Make the YAML section yours**: Add fields like `blog:`, `twitter:` or `pronouns:` to make it personal. YAML is flexible.
2. **Flat badges vs for-the-badge**: This template uses `style=flat-square` for a cleaner, smaller look. Change to `style=for-the-badge` for larger, bolder badges.
3. **Project table formatting**: Add more columns like `Tech` or `Status` if you want — markdown tables are easy to extend.
4. **Light-mode only**: This template looks best in light mode. If you want dark-mode support, add `theme=default_repocard` or switch to a dark theme.
5. **Remove the capsule header**: If you want even cleaner look, delete the entire `<div>` with the capsule-render img — the profile will still look complete.

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with your actual GitHub username
5. Repeat the find-and-replace for `your-name`, `your@email.com`, `<your-city>`, `<your-country>`, `<job-title>`
6. Update the YAML bio and projects table with your real info
7. Commit — your new profile is LIVE! 🎉
