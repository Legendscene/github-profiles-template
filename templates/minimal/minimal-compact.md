## 🔧 Customization

| Variable | Replace with |
|----------|-------------|
| `your-name` | Your display name |
| `yourusername` | Your GitHub username |
| `your-tagline` | Short 5-word tagline |
| `[current project]` | What you're working on |
| `[learning]` | What you're learning |

## ✨ Features

- Ultra-compact (fits on one screen)
- Stats + languages inline
- One-line status bar
- Compact language badges
- Single-row trophies
- Visitor counter + followers
- Vue theme (soft green)
