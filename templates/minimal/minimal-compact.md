## 🔧 Complete Customization Guide

### Step 1: Basic Information
| What to Replace | Where It Appears | Replace With | Example |
|----------------|-----------------|--------------|---------|
| `<your-name>` | Main heading | Your full name | Jamie Kim |
| `yourusername` | ALL URLs, stats, trophies, counters | Your GitHub username | jamiekim |
| `<your-tagline>` | Subtitle below name | Short tagline (5-7 words) | Full Stack Developer |
| `[current project]` | Status bar | What you're working on | building a SaaS platform |
| `[learning]` | Status bar | What you're learning | Kubernetes |
| `[ask me]` | Status bar | What you know about | React, Node.js |
| `yourwebsite.com` | Portfolio badge link | Your URL | https://jamiekim.dev |

### Step 2: Social Links
Replace the URLs in the first badge row:
- `https://github.com/yourusername` → `https://github.com/YOUR_USERNAME`
- `https://linkedin.com/in/yourusername` → `https://linkedin.com/in/YOUR_LINKEDIN`
- `https://twitter.com/yourusername` → `https://twitter.com/YOUR_HANDLE`
- `https://yourwebsite.com` → Your portfolio URL

### Step 3: Language Badges
Replace the example badges with your actual tech stack:
- Each badge: `![Name](https://img.shields.io/badge/Name-HEXCODE?style=flat&logo=logo-name&logoColor=white)`
- Find logo slugs at `https://simpleicons.org/`
- Add or remove badges to match your skills (aim for 4-8)

### Step 4: Stats Cards
Replace `yourusername` in both stats URLs and the trophy URL:
- `https://github-readme-stats.vercel.app/api?username=yourusername&...`
- `https://github-readme-stats.vercel.app/api/top-langs/?username=yourusername&...`
- `https://github-profile-trophy.vercel.app/?username=yourusername&...`

### Step 5: Theme Color
This template uses `theme=vue` (soft green). Change to any theme:
- `default` (blue), `tokyonight` (dark blue), `radical` (pink), `gruvbox` (orange/brown)
- Find the full list at `https://github.com/anuraghazra/github-readme-stats#themes`

### Step 6: Followers & Visitor Counters
Replace `yourusername` in:
- `https://komarev.com/ghpvc/?username=yourusername&...`
- `https://img.shields.io/github/followers/yourusername?label=Followers&style=social`

## 💡 Pro Tips

1. **Ultra-compact fit**: This design fits on a single 1080p screen. If you add more sections, keep them below the counters to maintain the compact feel.
2. **Inline status bar**: The `|` separators between `🔭 [project] | 🌱 [learning] | 💬 [ask me]` keep it tight. Add more items like `⚡ [fun fact]` by duplicating the pattern.
3. **Flat badge style**: Uses `style=flat` for minimal badges. Switch to `style=flat-square` for slightly taller badges, or `style=for-the-badge` for large bold ones.
4. **Single-row trophies**: The `row=1&column=6` params force a single row of 6 trophies. Remove `row=1` to allow multiple rows if you earn more trophies.
5. **Wave animation**: The waving hand GIF adds personality — keep it unless you want a truly text-only header. It loads from MartinHeinz's repo, so no setup needed.

## 🚀 One-Click Deploy

1. Copy everything inside the **"Ready-to-Use Code"** block above
2. Go to your profile repo: `github.com/YOUR_USERNAME/YOUR_USERNAME`
3. Open `README.md` and paste over everything
4. Press **Ctrl+H** (or **Cmd+H** on Mac) → find `yourusername` → **Replace All** with your actual GitHub username
5. Replace `<your-name>`, `<your-tagline>`, and bracket placeholders
6. Update the language badges and status bar
7. Commit — your new profile is LIVE! 🎉
