## 💡 Pro Tips

1. **Auto-update videos**: Add a GitHub Action using `lowlighter/metrics` or `youtube-to-markdown` to auto-fill your latest videos daily.

2. **Channel ID trick**: Your channel ID is NOT your handle. Check `https://youtube.com/account_advanced` or use `https://commentpicker.com/youtube-channel-id.php`.

3. **Sub count badge**: The subscriber badge may take 24h to update. It's cached by shields.io. Use `?cacheSeconds=3600` param to control refresh.

4. **Gear table sells**: Viewers love gear breakdowns. Link each item with an affiliate URL using `[Name](https://amzn.to/...)` format.

5. **Schedule consistency**: Put your upload schedule in the README so subscribers know when to expect content. This drives return traffic.
