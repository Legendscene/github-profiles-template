## 🔧 Customization

| Variable | Replace with |
|----------|-------------|
| `your-name` | Your name |
| `yourusername` | Your YouTube handle |
| `UCXXXX` | Your YouTube channel ID |
| `yourwebsite.com` | Your website |

## ✨ Features

- YouTube subscriber/views badges
- Latest videos section (auto-update with GitHub Actions)
- Content schedule info
- Gear/equipment table
- Patreon / Buy Me A Coffee links
- Twitch / Discord integration
